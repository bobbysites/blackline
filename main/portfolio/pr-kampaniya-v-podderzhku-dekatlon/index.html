PR-кампания в поддержку
Декатлон

193 публикации за 3 месяца
Охват 50 млн человек
PR Value 4,5 млн рублей

Декатлон – французская сеть спортивных гипермаркетов для всей семьи, основанная во Франции в 1976 году. В задачи агентства вошло формирование имиджа бренда как лидера спортивной индустрии через публикации в СМИ, привлекая к сотрудничеству максимальное количество изданий. Агентством был сгенерирован контент для публикаций без коммерческих вложений.


Мы сформировали релевантную базу СМИ, подходящую для целевой аудитории бренда по тематикам lifestyle, sport, news, retail. Подготовили новый креативный контент для формирования у СМИ нужного имиджа бренда. Наладили каналы коммуникаций, привлекли максимальное количество СМИ, заинтересованных в инфоповодах бренда. Инициировали публикации статей на различные темы, от актуальных товаров, до кадровых инициатив компании, тем самым раскрыли множество аспектов жизни бренда для разной целевой аудитории. Составили отчет о проделанной работе за каждый месяц.


За 3 месяца работы командой Агентства было инициировано 193 выхода в СМИ, среди которых публикации в изданиях первого эшелона: Эхо Москвы, РБК тренды, НТВ, Коммерсант, телеканал Дождь, Яндекс Новости, Рамблер женский. Таким образом, был поддержан имидж бренда, как лидера на рынке спортивной индустрии и как надежной ответственной компании, налажены каналы коммуникаций со СМИ, созданы и опубликованы наиболее актуальные и важные темы для всего мира.
